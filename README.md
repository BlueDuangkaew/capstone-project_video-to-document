# Video-to-Documentation Platform

This project converts instructional videos into structured documentation with screenshots, using AI models and computer vision.

## Features (planned)
- Video segmentation
- Whisper transcription
- Content filtering with Llama 3
- Keyframe extraction with OpenCV
- Auto documentation generation (Markdown, PDF, DOCX)
- Web interface (upload → preview → download)

## Tech Stack
- FastAPI, Python
- FFmpeg
- OpenCV
- Whisper
- GPT4All / Llama 3
- Docker (optional)

## Development Setup
1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Run FastAPI server
